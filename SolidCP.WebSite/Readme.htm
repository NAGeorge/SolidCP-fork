SolidCP Web Site - Readme

SolidCP Web Site hosts a web service used by SolidCP Installer for:

  • getting the information about available SolidCP components and their latest releases
  • getting update information for particular release
  • downloading SolidCP component distributives

The service stores the information about SolidCP components and their releases in "Data\ProductReleasesFeed.xml" XML file. This XML file could be downloaded by any external 3rd-party application.

SolidCP distributive installation and update files are stored in "Files" folder. Those files could be freely downloaded by any external client provided they know their exact URLs.

Project Structure

The project has the following folders structure:

  • \Trunk - project version currently in development
    • \Sources - project source codes.
  • \Releases - contains all project releases
  • \Branches - contains project branches for proof-of-concepts, new development, bug fixing, etc.
  • LICENSE.txt - project license
  • Readme.htm - the file you are reading right now
  • ReleaseNotes.htm - project release notes
